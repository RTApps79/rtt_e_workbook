Statistics Studio

A lightweight, generic statistics simulator for teaching simple data analyses. Paste a CSV file, choose variables, run common descriptive or inferential tests, and generate plain-language and APA-style interpretations. All calculations run locally in the browser.

2. Descriptive statistics

3. Frequency table

Summarize a categorical, ordinal, or text variable with counts, percentages, and a bar chart.

6. Chi-square test of independence

7. Likert survey summary

Summarize ordinal survey items. The scale average is intentionally simple and assumes selected items are scored in the same direction.

Scope and limitations

This tool is intended for classroom practice, lab exercises, basic QI-style summaries, and research-literacy instruction. It is not a full replacement for professional statistical software for high-stakes publication, complex models, large datasets, or regulated analyses. Results should be verified before formal use.
